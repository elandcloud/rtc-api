# rtc-api

A tool that runs microservices and their dependent services. These services are generated by docker-compose and can docker on linux, mac, windows 10, windows 7, etc.

## run cmd

[cmd](./cmd/README.md)

## run api

### set env

```sh
windows
$env:IS_RTC_API="Y"
mac
export IS_RTC_API="Y"
```

### run

```sh
$ go run .
```
